# Naming

## Files

| Type | Example |
|------|---------|
| REST response DTO | `product_list_response.dart` |
| REST request DTO | `product_detail_request.dart` |
| Domain entity | `product_list.dart` (no suffix) |
| List item widget | `card_product_item.dart` |
| Reusable widget | `avatar_widget.dart` |
| Controller | `login_controller.dart` |
| State | `login_state.dart` |
| Screen | `login_screen.dart` |
| Private section widget | `_login_section.dart` |

## Classes and Variables

Follow [Effective Dart: Style](https://dart.dev/guides/language/effective-dart/style). Keep naming patterns consistent within the project.
